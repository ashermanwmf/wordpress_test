== Installation ==
1. Upload `yellow-pencil-visual-theme-customizer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Wordpress Panel > Appearance > Yellow Pencil Lite
